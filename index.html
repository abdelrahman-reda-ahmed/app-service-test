Deployment Guide

You can deploy this application directly to Azure Static Web Apps without GitHub using one of these methods:

1

Azure Portal

Create a Static Web App resource in Azure Portal and upload the ZIP file containing this HTML file.

2

Azure CLI

Use the Azure CLI with the command: az staticwebapp create --name your-app-name --resource-group your-resource-group --source ./

3

Visual Studio Code

Use the Azure Static Web Apps extension for VS Code to deploy directly from your editor.
